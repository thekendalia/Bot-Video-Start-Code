## Installation
**IMPORTANT:** Rename ***REMOVE.env*** to just ***.env***

After you have it opened in your IDE run the below command

```bash
pip install -r requirements.txt
```

## Usage
Make sure you fill in both the TOKEN and WEBHOOK in the .env
```python
TOKEN=
WEBHOOK= 
```
